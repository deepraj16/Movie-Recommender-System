# 🎬 Movie Recommendation System

## Introduction

The **Movie Recommendation System** is a content-based recommender system trained on a dataset containing 5000 movies. It utilizes cosine similarity to suggest movies similar to a given input. This project includes an end-to-end Machine Learning pipeline with a GUI web application built using Streamlit.

## 🚀 Features

- **Content-Based Filtering** – Recommends movies using cosine similarity.
- **User-Friendly Interface** – Built with Streamlit for an interactive experience.
- **Movie Search** – Find recommendations based on a selected movie.
- **Efficient ML Pipeline** – Uses text preprocessing and feature extraction techniques.

## 🛠️ Tech Stack

- **Machine Learning:** Scikit-learn, NLTK
- **Data Handling:** NumPy, Pandas
- **Web Framework:** Streamlit
- **Similarity Measurement:** Cosine Similarity

## 🛠️ Installation and Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/movie-recommendation-system.git
cd movie-recommendation-system
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
streamlit run app.py
```

### 4️⃣ Open in Browser

Go to **`http://localhost:8501/`** to explore the recommendation system.
